# collision_checker
Checking the frequency collisions in the fixed-frequency linear-coupling superconducting qubit system

## Frequency collisions taken into account
- Type0A
- Type0B
- Type1A
- Type1B
- Type2A
- Type2B
- Type3
- Type7
- Type8
- Type9
