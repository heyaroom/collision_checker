# collision_checker
Checking the frequency collisions [[1]](https://www.nature.com/articles/s41534-021-00464-5) in the fixed-frequency linear-coupling superconducting qubit system

## Frequency collisions taken into account
- Type0A
Dead or Bad qubit
- Type0B
- Type1A
- Type1B
- Type2A
- Type2B
- Type3
- Type7
- Type8
- Type9
